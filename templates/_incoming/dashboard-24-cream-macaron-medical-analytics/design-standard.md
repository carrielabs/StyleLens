# Design Standard

## 风格定位

奶油马卡龙医疗数据分析看板，适合诊所运营、门诊效率、患者画像和财务盈余类 B 端分析场景。

关键词：温暖、柔和、医疗运营、数据标注、Hover 交互。

## 视觉规则

- 主背景使用奶油色 `#FAF3E0`。
- 卡片使用低饱和马卡龙色：柠檬黄 `#FCE79C`、蜜桃粉 `#F8C8DC`、雾霾蓝 `#BFE0F7`、薄荷绿 `#C1EBD6` 和白色 `#FFFFFF`。
- 卡片圆角以 `2rem` 到 `2.5rem` 为主，形成柔和 B 端看板风格。
- 卡片 Hover 使用轻微上浮和软阴影。

## 图表规则

- 图表使用 ECharts 5.5.0。
- Tooltip 统一为近白色背景、浅灰边框、12px 圆角和轻阴影。
- 柱状图、横向条形图、热力图、雷达图、漏斗图、瀑布图和面积图均已提供显式数据标签。
- 瀑布图 Tooltip 必须过滤辅助占位柱，只展示真实财务变动值。

## 本次入库检查

- HTML 是完整单页 Dashboard，可直接浏览器打开。
- 结构符合用户交付的 4 个 section 规划。
- 9 个图表 ID 与交付配置一致。
- 原始配置只列出 11 个文字槽位，实际 HTML 有 14 个可编辑槽位；入库版已按实际 HTML 补齐。
- 原始 HTML `lang="en"` 和页面标题为英文，入库版已规范为 `zh-CN` 和中文标题。
