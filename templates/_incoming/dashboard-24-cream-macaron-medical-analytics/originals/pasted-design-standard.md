# 奶油风 B端数据分析看板设计规范

## 1. 视觉语言与基调

- 温暖奶油底色：主背景使用暖黄调的 `#FAF3E0`，提供温暖有呼吸感的物理纸质质感。
- 低饱和马卡龙卡片：
  - 柠檬黄 `#FCE79C`：对应高亮、警示或关键分析点。
  - 蜜桃粉 `#F8C8DC`：对应紧急、敏感或核心医疗指标。
  - 雾霾蓝 `#BFE0F7`：对应常规、平稳指标。
  - 薄荷绿 `#C1EBD6`：对应健康、盈余、积极指标。
  - 纯白 `#FFFFFF`：作为中性平衡容器。
- 大圆角美学：卡片与模块一律使用 `rounded-[2rem]` 或 `rounded-[2.5rem]` 的大圆角，配合微妙的卡片悬浮过渡。

## 2. 数据标注规范

- 为了防止模板在未填充前显得空洞，所有图表必须开启显式数据标注。
- 横向/纵向柱状图：必须使用 `label: { show: true }`。数值必须紧贴在柱体顶部或右侧，采用等宽、高对比度的极深灰色。
- 环形图：外侧通过高对比度的数值与占比线明确指示，防重叠且优雅。

## 3. 看板级 Hover 交互与 Tooltip 规范

- 卡片悬停反馈：容器卡片在鼠标滑过时，应通过 `transform: translateY(-3px)` 产生微妙的垂直位移，并叠加软阴影。
- ECharts 悬浮框：统一替换为暖白色半透明容器 `rgba(255, 255, 255, 0.98)`，搭配超细灰色边框 `#EFEFEF` 及 `12px` 圆角。
- 阴影与对比度：利用微妙的向下发散阴影 `rgba(0,0,0,0.05)`，确保在马卡龙色背景上浮现。
- 高亮聚集：鼠标悬停在多折线、多柱体、多雷达维度时，设置 `emphasis: { focus: 'series' }`。
