# Design Standard

## 定位

现代 B 端数据分析 Dashboard 模板，采用浅色毛玻璃、轻量仪表盘和文件活动数据结构，适合存储用量、文件访问、运营活动和轻量级报告展示。

## 核心原则

1. 极简与通透：采用 `rgba(255,255,255,0.75)` 玻璃态卡片、`backdrop-filter: blur(20px)` 和浅色氛围背景。
2. 数据聚焦：核心指标使用高对比度数字和克制的文本层级，让首屏快速读取关键结果。
3. 轻拟物光影：使用浅蓝、浅橙弥散光晕营造高级数字空间感，不使用厚重阴影。

## 色彩

- 页面背景：`#F0F4F8`
- 卡片背景：`rgba(255, 255, 255, 0.75)`
- 主文本：`#111827`
- 次文本：`#6B7280`
- 弱文本：`#9CA3AF`
- 品牌蓝：`#3B82F6`
- 成功绿：`#10B981`
- 活力橙：`#F59E0B`
- 梦幻紫：`#8B5CF6`

## 字体

`Inter, -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif`

- 页面标题：30px，Bold，`#111827`
- 卡片标题：16px，Medium，`#374151`
- 核心指标数字：24px，Bold，`#111827`
- 辅助信息：11px - 14px，`#6B7280` 或 `#9CA3AF`

## 图表

- KPI Gauge：240 度开口半圆环，粗线条，圆角端点，百分比显示在图表内部。
- Trend Gauge：180 度半圆，细轨道线，中心使用气泡式标签。
- Segmented Bar：使用密集竖线条形码结构，通过透明度交错呈现分组质感。

## 交互

- 卡片保持稳定展示，不添加复杂 hover 动画。
- 可编辑文本聚焦时使用微弱蓝色底色提示。
- 图表数据在内联 ECharts 配置和 DOM 渲染函数中维护。
