# Design Standard

## 模板定位

面向管理层经营汇报、阶段性业务复盘和专项数据分析的企业经营分析报告模板。当前 HTML 实际采用暖色柔和报告风，适合偏轻量、友好的内部经营看板。

## 用户原始规范要点

- 风格关键词：专业、清晰、数据驱动、权威感。
- 页面结构：报告标题区、核心结论、3 个核心发现板块。
- 推荐色板：浅灰背景 `#F5F7FA`、白色卡片、深黑蓝文本 `#1F2D3D`、专业蓝 `#0052CC`。
- 卡片圆角：建议控制在 4px 到 8px。
- 图表要求：标题清晰、有单位、保留 hover tooltip、图例清晰。

## 当前 HTML 实际风格

- 页面背景：`#E4D8CD`
- 卡片背景：`#F5EFEB`
- 主文本：`#1A1A1A`
- 辅助文本：`#968F88`
- 强调色：`#FF9B70`、`#7AC794`、`#FFC656`、`#C8A2FF`、`#7EA9FF`
- 卡片圆角：24px，大于原始规范建议。

## 入库规范

- 可编辑文字统一使用 `data-editable="true"`、`contenteditable="true"`、`data-slot` 和 `data-key`。
- 图表容器统一使用 `id="chart-[name]"`。
- 原始交付文件保存在 `originals/`。
- 当前模板自动注入时以 `template.json` 的实际结构为准。
