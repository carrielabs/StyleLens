# 企业经营数据分析报告 - 设计规范

本设计规范用于企业级数据分析报告 HTML 模板，偏向周、月、季阶段性分析报告或经营复盘文档，不是实时监控大屏。

核心风格：专业、清晰、数据驱动、权威感。避免高饱和霓虹渐变、复杂毛玻璃、过度圆润卡片和无意义装饰背景。页面应像排版精良的报告文档，视觉服务于数据表达。

结构要求：报告标题区、核心结论、3 个核心发现板块；每个发现板块遵循“结论先行 -> 数据支撑”。板块间保留充足垂直间距，图表容器内部保留适当内边距。

推荐色彩：页面背景 `#F5F7FA`，卡片背景 `#FFFFFF`，一级标题 `#1F2D3D`，正文 `#3C4858`，辅助文本 `#8492A6`，边框 `#E5E9F2`，主强调色 `#0052CC`，增长 `#00A854`，风险 `#F04134`，警示 `#FFBF00`。

图表原则：简洁、保留 hover tooltip、图例清晰、标题和单位明确。禁止 Lorem ipsum、过多色彩、无标题无单位无交互图表、模棱两可的核心结论。
