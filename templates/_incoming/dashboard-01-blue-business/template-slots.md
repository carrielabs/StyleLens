# Template Slots

模板 ID：`dashboard-01-blue-business`

用途：数据分析报告、信息图、管理层看板。

## 模块

1. `hero`
   - 报告主标题与一句话结论。
   - 适合放报告主题、研究假设、核心判断。

2. `top-charts`
   - 顶部核心图表区。
   - 包含网络成本柱状图与增长箭头图。
   - 图表数据必须来自真实数据或明确标记为示例数据。

3. `content-drivers`
   - 成本驱动因素横向拆分。
   - 可隐藏。
   - 适合展示比例结构、成本构成、来源拆解。

4. `market-architecture`
   - 市场渗透率与架构对比。
   - 适合展示趋势变化、方案对比、技术路线切换。

5. `bottom-insights`
   - 结论与预测。
   - 适合放核心洞察、TAM 预测、行动建议。

## 注入规则

1. 所有 `data-editable="true"` 的文本可以进入页面内编辑模式。
2. 所有 `data-key` 对应 `template.json` 中的字段。
3. `window.REPORT_DATA` 是图表数据入口，后续真实数据应优先写入这里。
4. 不允许编造增长率、市场份额、金额、TAM 预测。
5. 没有真实数据时，应隐藏对应图表或明确使用示例标记。
6. 静态装饰元素不需要真实数据驱动。

## 外部依赖

当前模板依赖：

1. Tailwind CSS CDN
2. ECharts CDN

正式离线交付时，需要考虑本地化依赖或保留联网要求说明。
