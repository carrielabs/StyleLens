# RED CLAY 视觉设计标准

风格名称：Fine Art Red-Clay Brutalism

这套视觉体系采用红土海报美学，避免常见 AI 模板套路，例如大渐变、发光边框、弥散阴影和圆角卡片堆叠。

## 1. 色彩系统

1. 红土底色 `--c-clay: #B73E23`
   - 全站主背景。
   - 通过噪点层增加颗粒砂砾感。

2. 粉笔白 `--c-chalk: #FDF9F1`
   - 用于球场画线、主要文字、手画线框。
   - 避免纯白带来的电子冷感。

3. 网球荧光黄 `--c-neon: #D9EB26`
   - 只用于重点高光、序号前缀、Hover 网球实体。
   - 不允许大面积滥用。

4. 炭黑 `--c-net: #1C1514`
   - 用于球网网格和底部压重元素。

## 2. 质感规则

1. 使用双层噪点层制造沙粒和粉笔摩擦感。
2. 使用 `mix-blend-mode: overlay` 和 `mix-blend-mode: multiply`。
3. 使用 SVG 位移滤镜 `chalk-filter` 让线条产生粉笔手刷感。
4. 页面应保留粗粝、手作、海报式质感，不要变成干净 SaaS 风。

## 3. 交互规则

1. 网球鼠标追踪。
2. Hover 时网球光标放大。
3. 特训卡片 Hover 时出现击球残影。
4. 按钮点击时允许短暂形变。

## 4. 排版规则

1. 使用巨幅大字和强对比排版。
2. 大标题可以使用英文、短中文或中英混排。
3. 中文标题不宜过长，优先断行处理。
4. 正文控制在短段落，不适合大段连续说明。

## 5. 禁止事项

1. 不要加入大圆角卡片。
2. 不要加入普通 SaaS 阴影卡片。
3. 不要使用额外渐变背景。
4. 不要内联新增颜色破坏色彩系统。
5. 不要向模块里注入复杂嵌套容器。
6. 不要把这个模板用于严肃投研报告或密集数据看板。
