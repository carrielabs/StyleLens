# Lean Cyber-Analytics 设计系统标准

## 核心原则

数据导向、专业秩序、克制美学。避免刺眼多彩渐变、臃肿大圆角、模糊重投影和低信息密度。

## 色彩

- 主背景：`#070A08`
- 网格线：`#111813`
- 卡片背景：`#0D120F`
- 卡片边框：`#18221B`
- 主强调色：`#7CFF44`
- 次级激活色：`#1E351A`
- 主文字：`#FFFFFF`
- 次文字：`#889E8D`
- 弱文字：`#3E5244`

## 字体

- 英文/数字：`JetBrains Mono`, `SF Pro Display`, `ui-monospace`, `monospace`
- 中文：`PingFang SC`, `Microsoft YaHei`, `sans-serif`
- 看板标题：36px / 700
- 核心 KPI：40px-48px / 700
- 模块标题：14px-16px / 600
- 正文/副指标：14px / 400
- 轴线/微型文本：12px / 400

## 布局

- 核心间距：16px 或 20px
- 卡片圆角：12px
- 徽章圆角：9999px
- 卡片边框：1px
- Hover：只做轻微边框提亮或轻量亮度变化，不使用大范围发光。

## 图表

- 进度条轨道：`#18221B`
- 进度条填充：`#7CFF44`
- 折线：1.5px-2px
- 热力图：使用绿色透明度层级
- 环形图：暗绿轨道 + 荧光绿激活轨 + 中央白色百分比

## 交互

- 所有 `data-editable="true"` 元素必须同时带 `contenteditable="true"`。
- 聚焦态使用细描边，不能破坏整体视觉。
- Tab、Checklist、图表悬停等交互过渡控制在 150ms-200ms。
