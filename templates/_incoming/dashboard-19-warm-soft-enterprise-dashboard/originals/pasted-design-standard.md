# 数据分析 Dashboard / 数据报告 设计规范 - 暖色柔和版

## 1. 设计理念与定位

本模板采用“温暖智库”的设计语言，灵感来源于现代柔和拟态 UI。核心在于通过暖色调和柔和的光影，传递出友好、舒适但依然专业的数据阅读体验。

视觉关键词：暖色调、奶油风、柔和拟态、大圆角、亲和力。

适合场景：希望展现活力与温度的团队周报/月报、产品运营数据复盘、注重阅读体验的管理层看板。

## 2. 视觉与排版规则

核心色彩：

- 背景色：采用温暖的奶油色系，全局背景 `#F9F8F4` 或极淡的米色 `#FFFBEB`。
- 卡片背景：纯白 `#FFFFFF` 或极浅的暖灰 `#FAFAF9`。
- 主强调色：暖黄 `#F59E0B`、柔和橙 `#F97316`、珊瑚红 `#EF4444`。
- 辅助/图表色：偏暖的 Teal 青色、暖紫色、大地棕色系。
- 文本色：深色文本使用暖炭灰 `#4A4238` 或深褐灰色 `#57534E`；次要信息使用暖灰色 `#8C8272`。

阴影与质感：

- 使用扩散度大、透明度低、带有暖黄或暖橙色相的柔和弥散阴影。

排版与圆角：

- 卡片和图表容器采用 16px 到 24px 圆角。
- 字体优先使用圆润度较好的无衬线字体。

## 3. 模块与留白

保持宽松的版式布局，模块间距应大于模块内间距，确保视觉呼吸感。

## 4. 图表规范

- 图表配色严格遵循暖色调色板。
- 网格线使用极淡暖米色线条。
- Hover 时展现更明显的暖色光晕或轻微上浮效果。
