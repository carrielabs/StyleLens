# Design Standard

## 设计定位

- 风格：国际化企业级、专业克制、数据导向。
- 适合：管理层汇报、季度或年度经营分析、项目复盘、产品数据追踪。
- 不适合：重营销落地页、强视觉大屏、个性化插画页面。

## 视觉规范

- 背景：`#F8FAFC`，突出白色数据卡片。
- 卡片：`#FFFFFF`，弱阴影和浅边框。
- 主强调色：`#10B981`，用于增长、活跃状态和核心图表。
- 主文本：`#0F172A`。
- 次文本：`#64748B`。
- 警示色：`#EF4444`。

## 排版

- 字体：系统默认无衬线字体。
- 页面标题：24px 到 30px，字重 600 到 700。
- 模块标题：18px 到 20px，字重 600。
- 正文：14px 到 15px，行高保持舒适。
- KPI 数值：32px 左右，字重 700。

## 布局

- 模块间距：32px 到 48px。
- 卡片内边距：24px。
- 圆角：12px。
- 单行复杂图表不超过 3 个。

## 交互与图表

- 可编辑文本 hover 时提供轻微背景反馈。
- ECharts 图表配置 tooltip。
- 轴线、网格线保持克制，只保留必要参考线。
- 图例放在图表顶部或右侧。

## 入库说明

实际 HTML 基本符合设计规范。需要注意：交付说明建议采用 dataset 驱动，但当前 HTML 使用每个图表独立 `setOption` 的方式写入样例数据。
