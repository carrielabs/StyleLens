# 企业级数据分析 Dashboard 设计规范

## 设计定位与适用场景

- 风格基调：国际化企业级、专业克制、数据导向。
- 适用场景：管理层汇报、季度或年度经营分析、项目复盘报告、产品数据追踪。
- 不适用场景：重度营销页面、C 端强视觉冲击大屏、需要极度个性化插画的页面。

## 视觉规范

- 页面背景：`#F8FAFC`。
- 卡片背景：`#FFFFFF`。
- 强调色：`#10B981`。
- 主标题和核心数据：`#0F172A`。
- 正文和次要信息：`#64748B`。
- 下降或警告：`#EF4444`。

## 字体与排版

- 字体族：系统默认无衬线字体。
- 页面标题：24px / 28px，Font Weight 600。
- 模块标题：18px / 20px，Font Weight 600。
- 正文：14px，Line Height 1.6。
- KPI 数值：32px / 36px，Font Weight 700。

## 留白与圆角

- 模块间距：32px 或 48px。
- 卡片内边距：24px。
- 圆角：12px。

## 交互与图表

- 图表元素必须有 tooltip。
- 可编辑文本 hover 时应有极淡背景反馈。
- 图表去除繁杂网格线，仅保留必要水平参考线。
- 图例尽量位于顶部或右上角。

## 禁忌

- 禁止使用纯黑 `#000000` 作为大段文字颜色。
- 禁止使用超过 3 种完全不同的高饱和度色作为图表主色调。
- 禁止在同一行并排超过 4 个复杂图表。
- 禁止使用无业务意义占位符。
