# 国际范 B 端数据分析 Dashboard / 数据报告设计规范

本模板定位为中性冷灰底色、白色卡片、Emerald 高亮的企业级数据分析报告。适合高管决策看板、客户行为洞察、产品诊断报告和经营复盘。

## 视觉标准

1. 背景使用 `#F3F4F6` 到 `#F9FAFB` 的冷灰体系，入库版使用纯色 `#F3F4F6`。
2. 卡片使用 `#FFFFFF`、极浅边框和轻阴影，避免大面积渐变和重阴影。
3. 主色为 `#10B981`，深主色为 `#059669`；警告色 `#F59E0B`，风险色 `#EF4444`。
4. 标题和核心数值使用 `#111827`，正文使用 `#374151`，辅助文本使用 `#6B7280`。
5. 字体优先使用 Inter、Noto Sans SC、系统无衬线字体。

## 排版标准

1. 大数字使用紧凑字距，保持专业感。
2. 中文段落使用较高行高，保证可读性。
3. 模块间距保持 24px 左右，卡片内边距以 24px 为主。
4. 外层卡片统一使用 16px 圆角，内部元素使用更小圆角，避免同半径嵌套。

## 图表标准

1. 图表网格线保持低对比，不使用高对比实线。
2. 趋势图可使用低透明 Emerald 面积填充。
3. Tooltip 使用白底、深灰文字、细边框和轻阴影。
4. KPI 卡包含名称、数值、趋势和补充说明。

## 禁忌

1. 不使用大面积高饱和渐变背景。
2. 不出现 Lorem ipsum 占位符。
3. 不使用错误数字格式或拆断单位。
4. 不使用生硬闪烁交互。
5. 不让内容撑破卡片边界。

## 入库修正

1. 原始 HTML 的背景从轻微渐变调整为冷灰纯色。
2. 原始半透明卡片调整为白色卡片、浅边框和轻阴影。
3. 原始 `rounded-3xl` 外层卡片调整为 `rounded-2xl`。
4. 原始错误数值 `$16,00.72` 调整为 `8,234`。
5. 为静态 bar、funnel、table 模块补齐 `chart-*` id。
