# 原始设计规范归档

用户提供的设计规范摘要：

1. 定位：国际范 B 端数据分析 Dashboard / 数据报告模板。
2. 适用：企业级高管决策看板、客户行为洞察、产品与业务诊断报告。
3. 不适用：极简学术论文、移动端实时监控大屏。
4. 色彩：冷灰背景、白色卡片、Emerald 主色，警告 Amber，风险 Red。
5. 字体：Inter、系统无衬线字体、Microsoft YaHei。
6. 间距：模块 gap 约 24px，卡片内边距约 24px。
7. 圆角：大卡片 16px，小按钮和徽章 8px 或圆形。
8. 图表：浅虚线网格、低透明面积渐变、白底 Tooltip。
9. 禁忌：大面积渐变背景、Lorem ipsum、同半径圆角嵌套、错误数字格式、生硬交互。

入库版完整规范见上级目录 `design-standard.md`。
