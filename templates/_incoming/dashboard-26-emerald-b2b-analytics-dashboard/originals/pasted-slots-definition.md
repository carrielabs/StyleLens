# 原始槽位规范归档

用户提供的槽位规范摘要：

1. 核心模块：`key-takeaway`、`core-finding-1`、`core-finding-2`、`core-finding-3`。
2. 原始文字槽位命名建议：`text.header.title`、`text.header.subtitle`、`text.takeaway.title`、`text.finding_1.title` 等。
3. 原始图表类型声称支持：`kpi`、`area`、`column`、`donut`、`bar`、`funnel`、`radar`、`heatmap`、`waterfall`、`scatter`、`table`。
4. 实际 HTML 差异：HTML 使用 `text.header_title` 这类下划线槽位；实际未提供 `heatmap` 和 `waterfall` 模块。

入库版实际槽位见上级目录 `template-slots.md`。
