# Design Standard

## 模板定位

高级业务数据分析模板，目标质感是“真实产品官网质感”和“专业咨询报告质感”。整体应避免浮夸渐变、多余阴影、过度圆角和装饰性堆叠，优先保证数据阅读效率。

## 色彩规范

- 背景色：`#f5f6f7`
- 卡片背景：`#ffffff`
- 边框：`#dee0e3`
- 主文本：`#1f2329`
- 次文本：`#646a73`
- 主品牌色 / 核心业务：`#3370ff`
- 对比色 / 竞品：`#8f959e`
- 增长 / 积极数据：`#2fa86d`
- 风险 / 预警数据：`#d83931`
- 分类补充色：`#e89531`、`#635db1`

## 排版与布局

- 字体：`PingFang SC, Microsoft YaHei, sans-serif`
- H1 页面主标题：24px / 28px，加粗
- H2 卡片模块标题：16px / 18px，加粗
- 小标题 / 数据标签：14px
- 正文 / 辅助文字：12px
- 核心大数字：28px-36px，粗体
- 间距：遵循 4pt / 8pt 栅格
- 卡片内边距：20px 或 24px
- 卡片间距：16px 或 24px

## 组件规范

- 卡片：白底、细边框、6px-8px 微圆角，不使用弥散阴影。
- 洞察栏：使用极浅灰背景和细边框，突出结论，不做装饰性大色块。
- 表格：极简水平分割线，表头浅灰底，无垂直分割线。
- 状态标签：低饱和底色配同色系深色文字，避免高饱和纯色块。
- 图表：统一通过底部 `initCharts()` 管理 ECharts 配置，便于后续替换 option 数据。
