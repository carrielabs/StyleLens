# Premium Dark Dashboard Standard

基于 Spotify Dark 视觉方向，参考企业级数据分析看板，目标是高信息密度、专业客观、低 AI 感。

## 色彩

- 基础背景：`#09090B`
- 容器背景：`#121214`
- 边框：`rgba(39, 39, 42, 0.4)`
- 主要文字：`#F4F4F5`
- 次要文字：`#A1A1AA`
- 默认强调色：`#1DB954`
- 异常警示：`#F43F5E`

避免大面积多彩渐变、厚重阴影和过饱和背景。暗色层次主要靠细边框和轻微背景差异建立。

## 字体

- 英文/数字：Inter, system-ui
- 中文：PingFang SC, Microsoft YaHei
- H1：24px / bold
- 模块标题：16px / semibold
- KPI 数值：32px 或 36px / extrabold
- 标签与辅助文字：10px-12px

## 布局

- 页面主容器最大宽度：1600px
- 桌面边距：24px-32px
- 网格：12 列
- KPI 区：4 列
- 主分析区：左 7/12，右 5/12
- 间距：16px 或 24px

## 质感

- 卡片圆角：6px-8px
- 卡片边框：1px 半透明深灰
- 图表水平网格线保持弱化
- 折线图使用低透明填充
- 柱状图使用细条和小圆角
