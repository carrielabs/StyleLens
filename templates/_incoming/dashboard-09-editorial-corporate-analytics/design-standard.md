# 视觉风格设计标准 (Editorial Corporate Style)

## 1. 颜色规范 (Color Palette)
保持绝对的克制，避免使用超过两种品牌主色。不要使用任何大面积渐变色。
- **页面底色 (Page Background):** `#f4f5f6` (高级浅灰，凸显内容卡片的质感)
- **卡片底色 (Card Background):** `#ffffff` (纯白，用于常规数据) / `#0a0a0a` (纯黑，用于强调结论)
- **品牌主色 (Brand Primary):** `#25a858` (咨询绿，用于核心数据和正向指标)
- **品牌次色 (Brand Secondary):** `#1b8544` (深绿，用于层次拉开)
- **文本颜色 (Text):** `#1f2329` (主标题、正文) / `#646a73` (辅助说明、图注)
- **边框与分割线 (Border):** `#dee0e3` (极细，用于隐形网格)

## 2. 字体与排版层级 (Typography)
放弃手写体或圆体，使用无衬线字体（如 Inter, Helvetica, PingFang SC），利用绝对的字号和粗细对比来引导视觉。
- **超大数值 (Metric Value):** `64px`, `Font-weight: 700`, 行高 `1.1` (用于核心数据点)
- **主标题 (H1):** `32px`, `Font-weight: 700`, 行高 `1.2` 
- **模块标题 (H2):** `24px`, `Font-weight: 700`
- **卡片标题 (H3):** `18px`, `Font-weight: 600`
- **正文与小注 (Body/Small):** `14px` / `12px`, 颜色弱化

## 3. 留白与间距 (Spacing System)
模块之间的呼吸感是“高级感”的来源，采用 8px 倍数系统。
- **模块间距:** `16px` (紧凑) / `24px` (常规网格)
- **卡片内边距:** `32px` 或 `48px` (提供充足的留白，不要拥挤)

## 4. 图形与圆角 (Shapes & Radius)
- **彻底摒弃“AI味”:** 拒绝大范围的 Glassmorphism（玻璃拟态）、发光阴影或巨大的圆角（>16px）。
- **微圆角 (Border Radius):** 统一使用 `4px - 6px`。边缘几乎是直角，带来严谨、理性的商业质感。
- **阴影 (Shadow):** 极度克制，仅在白色卡片上使用 `0 2px 4px rgba(0,0,0,0.02)`，几乎不可见，仅为和灰色背景拉开层次。
