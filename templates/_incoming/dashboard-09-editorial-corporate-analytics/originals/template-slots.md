# 页面模块槽位 (Template Slots)

所有模块均支持 `data-section` 标识，模块内部可替换的文案均带有 `data-editable="true"` 属性。

## 1. 报表表头槽位 `data-section="header"`
- **用途:** 放置报告的主标题、副标题（报告核心结论的一句话总结）、以及生成日期/数据源时间。

## 2. 核心摘要槽位 `data-section="key-takeaways"`
- **用途:** 提炼自 Excel 的前 3 个最重要指标。
- **视觉:** 三栏等宽网格（Grid-3），由主品牌色（绿）、次品牌色（深绿）和黑色卡片组成，对比度极高，第一时间抓住眼球。

## 3. 占比/对比分析槽位 `data-section="chart-split-bar"`
- **用途:** 适用于 Excel 中的分类汇总数据（例如：“是否继续投资”的选项分布）。
- **视觉:** 左右对半排版。左侧大黑块输出核心洞察；右侧白底使用原生 CSS 实现的高质量横向柱状图（Bar Chart），摆脱对 Echarts/Chart.js 等重型框架的依赖。

## 4. 单一指标强调槽位 `data-section="chart-split-donut"`
- **用途:** 适用于 Excel 中需要极致强调的某个核心百分比数据（例如市场占有率、完课率等）。
- **视觉:** 左侧洞察文案，右侧极简的 CSS 半环形图（Donut Chart），具有极强的冲击力。

## 5. 漏斗/人群分层槽位 `data-section="archetypes"`
- **用途:** 将数据对象分类（例如：高频用户、中频用户、低频用户），或是流程步骤。
- **视觉:** 横向带有虚线轴线的时间线/步骤卡片，展示各群体的体量占比及行为特征。

## 6. 数据明细下钻槽位 `data-section="data-table"`
- **用途:** 针对数据报告，通常需要下钻一层看具体业务线的表现（Excel 透视表的结果展示）。
- **视觉:** 专业干净的表格 (Data Table)，包含底部边框对齐，并使用 Status Badge（状态标签）标示结论，非常符合字节/腾讯等公司的汇报习惯。
