# Design Standard

## 定位

企业级数据分析 Dashboard 模板，Dark Mode，高信息密度，适合直播、电商、流量、经营复盘和投放效果追踪。

## 核心原则

1. 数据至上：视觉服务于数据，不使用干扰阅读的大面积渐变、厚重阴影或夸张圆角。
2. 高信息密度：紧凑网格布局，首屏展示核心指标和主要趋势。
3. 模块化克制：通过 1px 边框和小间距区分层级。

## 色彩

- 页面背景：`#141414`
- 模块背景：`#1F1F1F`
- 边框：`#333333`
- 主文本：`#E5E5E5`
- 次文本：`#999999`
- 琥珀黄：`#EAB308`
- 青海绿：`#14B8A6`
- 科技蓝：`#3B82F6`
- 紫色：`#8B5CF6`
- 玫瑰红：`#F43F5E`

## 字体

`-apple-system, BlinkMacSystemFont, "PingFang SC", "Microsoft YaHei", sans-serif`

- 大标题 / Key Takeaway：18px - 24px，字重 600。
- 模块标题：14px - 16px，字重 500。
- 核心指标数据：20px - 28px，使用 `font-variant-numeric: tabular-nums;`。
- 正文 / 图表标签：12px，字重 400。

## 间距与交互

- 外边距：24px 或 16px。
- 内部间距：8px 或 12px。
- 图表卡片内部 Padding：16px。
- Hover 仅用于可编辑文本和图表 Tooltip。
- Tooltip 使用深色半透明背景，和主题保持一致。
