# 企业级数据分析 Dashboard 模板设计规范 (Dark Mode)

## 1. 核心设计原则

数据至上 (Data-Driven First)：视觉服务于数据，摒弃一切干扰数据阅读的装饰（如渐变色背景、大面积弥散阴影、夸张的圆角）。

高信息密度 (High Density)：面向专业复盘与管理层汇报，采用紧凑的网格布局，确保首屏能展示尽可能多的核心指标与趋势。

模块化与克制 (Modular & Restrained)：通过严谨的 1px 边框和极小的模块间距（Gap）划分信息层级，避免视觉松散。

## 2. 适用与不适用场景

适合的报告类型：直播/电商单场经营复盘、大盘流量监控看板、月度/季度业务分析报告、广告投放效果追踪。

不适合的报告类型：面向 C 端用户的年度账单、需要大量长图文排版的品牌宣发文章。

## 3. 色彩规范

- 页面背景：`#141414`
- 卡片/模块背景：`#1F1F1F`
- 边框：`#333333`
- 主文本：`#E5E5E5`
- 次文本：`#999999`
- 主色 1：`#EAB308`
- 主色 2：`#14B8A6`
- 主色 3：`#3B82F6`
- 辅助色：`#8B5CF6`、`#F43F5E`

## 4. 字体排版与中文处理

字体族：`-apple-system, BlinkMacSystemFont, "PingFang SC", "Microsoft YaHei", sans-serif`

- 大标题/Key Takeaway：18px - 24px，字重 600。
- 模块标题：14px - 16px，字重 500。
- 核心指标数据：20px - 28px，使用等宽数字字体。
- 正文/图表标签：12px，字重 400。

中文排版禁忌：禁止在中英文之间漏加空格；禁止在数据单位前换行；禁止段落文本两端不齐。

## 5. 交互与留白

- 外边距统一使用 24px 或 16px。
- 内部元素间距使用 8px 或 12px。
- 图表卡片内部 Padding 为 16px。
- Hover 仅在图表 Tooltip 和可点击筛选/导出按钮上使用。
- 图表 Tooltip 必须使用暗色半透明背景，带轻微毛玻璃效果。
