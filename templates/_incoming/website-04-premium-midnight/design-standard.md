# Premium Midnight 高端暗夜官网模板设计标准

## 核心定位

本模板锁定 StayInn Premium Midnight Mode，服务于高端数智化 SaaS、空间资产管理、硬科技官网和政企汇报型产品页。

## 视觉原则

1. 全站固定暗色背景：`#09090B`。
2. 主卡片使用低饱和炭黑：`#121214` / `#18181B`。
3. 边框使用极细灰色：`#27272A`。
4. 主文本使用珍珠白：`#F4F4F5`。
5. 副文本使用银灰：`#9A9A9E`。
6. 重点状态使用靛蓝 `#4F46E5`、极客紫 `#7C3AED`、安全绿 `#10B981`。

## 模块特征

1. `product-showcase`：拟真系统操作台，带 Tab 切换，展示房态、派单、收益精算。
2. `architecture`：三段式系统拓扑，展示源端、云端、终端链路。
3. `value-prop`：同心圆收益占比图。
4. `metrics`：月度活跃柱状图与生态气泡数据切换。
5. `problems-solutions`：错落痛点诊断卡片。
