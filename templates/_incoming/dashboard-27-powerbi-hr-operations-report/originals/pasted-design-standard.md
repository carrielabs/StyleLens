# 企业级数据分析 Dashboard 设计规范 (Power BI 商务精简版)

## 1. 视觉设计原则 (Visual Principles)

去 AI 塑料感：拒绝大面积的渐变色背景和过度的阴影。整体采用微弱阴影（shadow-[0_2px_8px_rgba(0,0,0,0.04)]），卡片圆角严格控制在 4px，保持 Power BI 级别的硬朗专业工业质感。

高信息密度布局：不妥协于过宽的间距。网格间距固定为 12px 或 16px，卡片内边距使用紧凑的 12px - 16px，最大化一屏内的数据展现效率。

精炼页眉设计：去除了动态 Timeline 历史快照导航与多余的重置控件，采用纯单页式管理报告（Single-page Operational Report）形态，使管理者更聚焦于当期核心指标。

背景与容器对比：

全局背景色采用低饱和度、高级中性的淡蓝灰：#EBF1F6。

顶部控制面板采用深海军蓝：#2C3E50，文字采用高对比度的纯白或淡蓝色。

数据卡片容器采用纯白：#FFFFFF，配以极细的描边 #DDE3EA。

## 2. 颜色体系 (Theme Palette)

主色 (Navy): #2C3E50 (控制面板、主标题、图表重色轴)

辅色 (Slate): #475569 (普通文字、未激活状态)

装饰金 (Gold): #D97706 或 #EAB308 (用于 Retention 核心资产高亮、折线图关键点)

性别色 (Gender): 男士 #2C3E50 (稳重深蓝)；女士 #C0392B (优雅暗红)

图表色系: 统一使用商务冷暖色调结合的多色阶循环：#2C3E50（深蓝）、#EAB308（亮金）、#E67E22（深橙）、#C0392B（暗红）、#27AE60（青绿）、#7F8C8D（石板灰）。

## 3. 中文排版与数字格式 (Typography)

系统字体: Inter, -apple-system, BlinkMacSystemFont, Microsoft YaHei, sans-serif。

数据字体: 核心指标值优先使用等宽、抗锯齿的 Inter 或 Segoe UI，确保多位数对齐时视觉美感。

数值规范: K/M 单位大写（如 $115.4K），百分比保留两位小数（如 89.79%），保持专业金融/管理报表标准。

## 4. 动态交互禁忌 (Interaction & Performance)

切换性能: 页面存在多页签（Tabs）或多视图切换（Chart/Table）时，必须在 DOM 变为可见后立即调用 ECharts 的 resize() 方法，否则会导致图表计算出 0 宽度而无法渲染。

过渡平滑: 所有的 Hover 效果（按钮、卡片、表项）仅允许微弱的位移或背景色淡入，过渡时间固定为 150ms，禁止使用大范围的缩放或剧烈旋转动画。
