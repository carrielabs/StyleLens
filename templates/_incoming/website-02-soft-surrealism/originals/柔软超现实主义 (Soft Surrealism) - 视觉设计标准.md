柔软超现实主义 (Soft Surrealism) - 视觉设计标准

该设计体系灵感源自于复古超现实主义绘画与胶片艺术，其核心灵魂在于“消灭一切锐利的边缘与生硬的几何体”。通过大面积高饱和度色块的极度模糊、互相晕染，营造一种如梦境般、在空气中拉丝流转的高级视觉体验。

1. 核心设计哲学

彻底消灭硬边缘: 页面中禁止出现常规的矩形色块、生硬的 1px 细线、常规的 Border边框 或具象物体。

色彩作为光晕: 所有的图形模块（如图标、背景装饰），必须转化为高斯模糊数值极高（Blur 30px~100px）的椭圆形（Oval），模拟色彩在空气中的晕开。

物理世界的胶片温度: 必须在全局覆盖一层低透明度（Opacity: 0.15）且混合模式为 overlay 的 SVG 颗粒噪点，这是区分“廉价网页”和“艺术品”的关键。

2. 调色板 (Color Palette)

拒绝大面积单调渐变，使用高饱和纯色在空间中的碰撞与融合：

晴空蓝 (Sky Blue): #1d70b8 - 主基调，代表无垠苍穹。

梦境粉 (Cloud Pink): #f19ebc - 核心点缀色，模拟云雾、残影，提供梦幻感。

草地绿 (Court Green): #189e53 - 用于提供强烈的冷暖对比与生命力。

亮黄点缀 (Ball Yellow): #eafb47 - 极少使用，仅用于局部提亮高光或融合（Mix-blend）。

宣纸白 (Soft White): #f8f7f2 - 所有文本颜色，带有微暖调，不刺眼。

3. 排版与字体 (Typography)

无衬线为主 (Sans-Serif): 正文和描述文本使用系统原生最干净的无衬线体（-apple-system, PingFang SC），字重偏轻（Light 300），拉大字间距（letter-spacing: 0.05em），让文本“漂浮”在页面上。

衬线大标题 (Serif Display): 针对超大字号的核心标题，混合使用经典的衬线体（如 Songti SC），增强艺术展的呼吸感与文化厚度。

4. 关键 CSS 艺术手法

动能拉丝擦除 (Motion Smudge):
使用绝对定位的伪元素或横向极度拉伸的色块（scaleX(1.5) + blur(20px)），模拟色彩高速运动时产生的横向拖影。

多重光晕融合 (Blend Modes):
当多个模糊椭圆重叠时，必须配合 mix-blend-overlay 或 mix-blend-screen，让色彩交界处产生剔透的空灵感，而非浑浊的泥巴色。

胶囊按钮 (Capsule Buttons):
所有交互按钮采用极度柔软的 border-radius: 9999px，交互时伴随柔和的颜色发光（box-shadow），没有硬框线。
