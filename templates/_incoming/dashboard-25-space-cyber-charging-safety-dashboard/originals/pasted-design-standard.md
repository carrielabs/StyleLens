# 智能硬件与充电安全大屏数据可视化设计标准

本标准指导“电动自行车充电站 AI 智慧安全监管平台”以及类似智能硬件物联网安全监控、社区网格化治理、企业安全资产负荷大屏等 HTML 模板设计。

## 适用场景与限制

极度适合：物联网实时监控大屏、城市治理/网格化看板、高频业务复盘与分析报告。

不适合：纯文本学术论文、电商大促热度看板、娱乐性营销活动。

## 色彩系统与视觉深度

- `#020617`：背景底色。
- `#07142b`：卡片填充色 / 30% 虚化遮罩。
- `#06b6d4`：核心高亮与主数据色。
- `#10b981`：安全与正常运行状态色。
- `#f59e0b`：注意与低优先级警告色。
- `#ef4444`：火灾、严重告警与极度危险色。

## 阴影与辉光

- 卡片边框辉光：`0 0 15px rgba(6, 182, 212, 0.15)`。
- 文字/数字高亮辉光：`text-shadow: 0 0 10px rgba(6, 182, 212, 0.5)`。
- 故障/警报状态辉光：`text-shadow: 0 0 12px rgba(239, 68, 68, 0.6)`。

## 字体与排版

- 英文/数字优先采用等宽字体或等高无衬线体。
- 中文使用 PingFang SC、Microsoft YaHei、Noto Sans CJK SC。
- 禁止正文小于 12px。
- 单行中文标题避免超过 12 个字。

## 指标卡与图表规则

- KPI 使用 24px-36px 大字号数据和 12px 标签。
- 指标数值必须带单位后缀。
- 折线图使用平滑曲线和半透明面积填充。
- 柱状图使用圆角。
- 图表网格线使用低透明虚线。

## 交互行为

- Hover 时卡片边框从弱青色变为强青色并扩大辉光。
- Tooltip 使用深色半透明背景和青色边框。
- 时间维度切换使用暗到明的过渡动画。
